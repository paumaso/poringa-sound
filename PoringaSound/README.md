# 🎧 Frontend - Plataforma Musical

Aquest és el frontend de la plataforma de música interactiva. Està desenvolupat amb **React** i utilitza **Vite** com a bundler.

---

## 🚀 Tecnologies

- React
- Vite
- TailwindCSS (opcional)
- Axios
- React Router

---

## 📦 Requisits

- Node.js (v18+ recomanat)
- npm

---

## 🛠️ Instal·lació

```bash
cd frontend
npm install

---

## 🚀 Ejecucion
npm run dev
